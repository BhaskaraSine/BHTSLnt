# BHTSLn't

BHTSL aims to fix some of the bugs with the Hypixel Housing module HTSL, as well as add some new features.
BHTSLn't is just my own random add-ons, I might abandon so don't expect too much.

## Installation instructions:
1. Download zip from releases
2. Extract zip to `./minecraft/config/ChatTriggers/modules/BHTSL/`
3. Launch game or run `/ct reload`

To edit import files, do `/ct files` ingame, navigate into the BHTSL folder, go into `imports` and you can create new importable files with the `.htsl` file extension.
To see ingame commands do `/bhtsl` or `/bhtsl help`.
